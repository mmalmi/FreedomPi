Freedom Pi

  • This is an open wifi hotspot that lets you communicate and share files with nearby people.
  • If Internet access is available, it is routed over Tor, which can bypass some censorship and protect the hotspot operator from liability.
  • Offline Library. A small version of Wikipedia is included by default, and other datasets can be added.
  • File listing
  • If a playlist exists, the hotspot broadcasts on the radio frequency 90.0 MHz. Low power transmission is unregulated in many countries, but do check your local regulations. If you want more power, set up an antenna.
  • If you move between multiple Freedom Pi hotspots, your browser synchronizes the chat messages between them. It's a public chat, so don't say anything confidential.
    • For private, non-anonymous, offline-first communication, check out Iris Messenger
